▶ HOW IT WORKS // 3 STEPS

[01]
Choose your error code

Browse 35 games — one per HTTP error code. 404, 500, 502, 503 and everything in between.

[02]
Buy and download

Instant download after payment. One zip. Unzip — you get standalone HTML files, one per game. No build step, no npm, no framework.

[03]
Point your server at it

One line in your config and you're done. Works with nginx, Apache, Caddy, Traefik — any server that supports custom error pages.

[ CLICK TO COPY ] [ COPIED! ] # nginx — add to your server block
error_page 404 /errors/404-maze-runner.html;
error_page 500 /errors/500-asteroid-field.html;
error_page 502 /errors/502-gateway-intruders.html;
error_page 503 /errors/503-missile-defense.html;

# Apache — add to .htaccess or VirtualHost
ErrorDocument 404 /errors/404-maze-runner.html
ErrorDocument 500 /errors/500-asteroid-field.html
ErrorDocument 502 /errors/502-gateway-intruders.html
